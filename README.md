# League of Heros

## Objetivo

Construir uma aplicação que possa adicionar Heros na melhor liga de todas, a sua. E colocar em prática o conteudo estudado no curso de react.

## Tecnologia

- React
  - react router dom
  - context API
  - react icon
- Typescript
- Html
- CSS/Sass com Modules
- Json-Serve
